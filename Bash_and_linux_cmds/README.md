# Bash and Linux basic Commands

* Where am I?
```pwd```

* How to create a directory?
```mkdir```

* How to create a file
```touch name of file```

* How to change directory 
```cd directory_name```

* How to come out of a directory
```cd ..```

* How to change file
```uname```

* How to list files
```ls```

* How to list all files
```ls a```
